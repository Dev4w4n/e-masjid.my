Logout Functionality Test

Test Steps:

  1. Go to http://localhost:3000
  2. Sign in with valid credentials
  3. Navigate to different protected pages (e.g., /profile, /admin)
  4. Click the logout button in the user menu
  5. Verify you are redirected to the login page
  6. Try to access protected pages directly (e.g., /profile)
  7. Verify you are redirected to login page on all protected routes

Quick Test Links:

Expected Behavior:

  • ✅ Sign out should clear the session token
  • ✅ User should be redirected to /auth/signin immediately
  • ✅ All protected routes should redirect to login when accessed directly
  • ✅ No authentication tokens should remain in browser storage
  • ✅ Refreshing any page should not auto-login the user
